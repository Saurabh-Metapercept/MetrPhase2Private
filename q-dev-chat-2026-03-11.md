**For converting Figma designs to code optimally:**

1. **Export assets from Figma:**
   - Use Figma's "Inspect" panel (right sidebar) to get exact CSS values
   - Copy colors, spacing, font sizes, border radius directly
   - Export icons as SVG or use icon libraries like Lucide

2. **Use Figma plugins:**
   - "Figma to Code" - generates HTML/CSS/React
   - "Anima" - exports to React/Vue with Tailwind
   - "Builder.io" - visual to code conversion

3. **Manual approach (most accurate):**
   - Take screenshots and place them side-by-side with your browser
   - Use browser DevTools to adjust values in real-time
   - Copy exact spacing (px values) from Figma's Inspect panel
   - Match colors using hex codes from Figma

4. **Best practice:**
   - Build component by component, not entire page at once
   - Use Tailwind's arbitrary values for exact Figma measurements: `w-[268px]` instead of `w-64`
   - Test responsiveness early if design has multiple breakpoints

The approach I used here (reading your detailed spec) is actually the most reliable for pixel-perfect results.

```typescript
// src/pages/DocMigration.tsx
const allMigrationTypes = [
  { id: 'docx-to-dita', name: 'DOCX to DITA', icon: '...' },
  { id: 'html-to-dita', name: 'HTML to DITA', icon: '...' },
  { id: 'md-to-dita', name: 'Markdown to DITA', icon: '...' }

function DocMigration() {
  const { user } = useContext(UserContext);
  
  return (
    <div>
      {allMigrationTypes.map(type => {
        const isEnabled = user.enabledMigrations.includes(type.id);
        
        return (
          <MigrationCard
            key={type.id}
            {...type}
            enabled={isEnabled}
            onClick={() => isEnabled ? startMigration(type.id) : showUpgradeModal()}
          />
        );
      })}
    </div>
  );
}
```

## 3. **Migration Card Component**

```typescript
// src/components/MigrationCard.tsx
function MigrationCard({ name, icon, enabled, onClick }) {
  return (
    <div

      onClick={enabled ? onClick : undefined}
    >
      <img src={icon} />

          <Lock size={20} /> {/* Show lock icon */}

    </div>

}
```
